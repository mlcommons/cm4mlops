# About
## Run command
```
cm run script --tags=app,mlperf,generic-harness_resnet50
```

To use any custom onnx model file we can do 
```
cm run script --tags=app,mlperf,generic-harness --modelpath=<CUSTOM_MODEL_FILE_PATH>
```

### Other Input Options:
* `output_dir`
* `scenario`
* `runner`
* `concurrency`
* `ep`
* `intraop`
* `interop`
* `execmode`
* `modelpath`
